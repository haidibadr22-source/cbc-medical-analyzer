
#  CBC Medical Report Analyzer (AI-Powered)

An AI system that extracts structured medical information from CBC (Complete Blood Count) PDF reports using **LangGraph, LLMs, and Gradio**.

The system:
- Extracts structured JSON from medical PDFs
- Validates extracted medical data
- Generates a clinical summary
- Provides a downloadable JSON output

# 📌 Features

### 📄 PDF Processing
- Upload CBC laboratory reports in PDF format
- Extract text using PyMuPDF

### 🧠 AI Pipeline (LangGraph)
- Parser Agent → Extracts structured JSON
- Validator Agent → Checks correctness
- Summary Agent → Generates medical summary

### 📊 Outputs
- Structured JSON output
- Validation report
- Medical summary
- Downloadable JSON file
